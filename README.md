<h3>Today I Learn 6차</h3>

<li>오늘 TIL 3줄 요약</li>
자신이 만든 프로그램은 사람들에게 영향을 준테니 완벽하게 책임을 져라.
코드를 살아있는 생명체처럼 대하라. 방치하면 죽는거와 똑같다. 

<li>책에서 기억하고 싶은 내용을 써보세요.</li>

<strong>클린코드란?</strong>
설명이 필요없는 코드. 코드를 읽기만 해도 이 코드가 무슨 일을 하는지, 어떤것을 의미하는지 물어볼 필요도 없이 스르륵 이해되는 그런 코드.

<strong>클린코드 꿀팁</strong>
1. 함수 이름을 짐작이 가능하도록 이름 지어라. 
2. 함수 이름은 무조건 동사로 지어라.  함수의 액션은 1개여야 한다. 하나의 함수는 1개의 역할만 하는것이 좋다. 그래야 함수가 하는 역할을 명확하게 지정할수 있다. 
3. 매개변수는 3개 이하로 써라. function 매개변수(1,2,3)
4. 불린값을 인자로 쓰지마라. 불린은 참,거짓에 따라 2가지 일을 처리해야하는데 함수는 1가지 일만 잘해야 한다.
5. 나 혼자만 알아볼수있는 축약어를 쓰지마라. 
6. 마지막에는 항상 위와같이 클린하게 코드 다듬는 작업을 해라.

코드도 살아있는 생명체처럼 죽지않도록 애정을 가지고 관리하고 개선해 나가야하다니...물줘야겠다.
